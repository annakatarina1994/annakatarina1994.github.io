Java && Spring Projects

Time To Project

A single page application prototype built with a small group of fellow developers, it allows users to create events that they can then invite friends to attend. It also allows you to compare your calendar with a friend's to easily see when you both have free time. Created using JavaScript, HTML and CSS on the front end combined with Java and Spring on the backend.
